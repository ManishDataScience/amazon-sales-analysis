# 📊 Amazon Sales Data Analysis

## 📌 Overview
This project analyzes Amazon sales data to extract meaningful business insights.

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Key Insights
- Identified top-selling product categories
- Analyzed revenue by state and size
- Evaluated order status (Delivered vs Cancelled)
- Found monthly and weekly sales trends

## 📈 Visualizations
- Bar charts
- Pie charts
- Line charts
- Heatmap

## 🚀 Conclusion
This project helps understand customer behavior and improve business decisions.
